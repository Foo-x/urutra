# うるおいトラッカー (Urutra)

シンプルな水分補給管理Webアプリケーションです。
日々の水分摂取量を記録し、波のアニメーションで視覚的に進捗を確認できます。

## 機能

- **簡単記録**: 120ml, 250ml, 330ml, 500ml のプリセットボタンで素早く記録。
- **カスタム入力**: スライダーで任意の量（10ml単位）を追加可能。
- **視覚的フィードバック**: 摂取量に応じて波の高さが変化。
- **データ保存**: リロードしてもデータが消えないローカルストレージ保存。
- **PWA対応**: ホーム画面に追加してアプリのように使用可能（オフライン対応）。
